# Project Overview

This project is a simple login page built with HTML and styled using Tailwind CSS along with Google Fonts and Font Awesome icons. It provides a user-friendly interface for users to log in to their accounts. The main features include fields for email and password, a "Remember me" checkbox, and a link for password recovery.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/yourproject.git
   ```

2. Navigate to the project directory:

   ```bash
   cd yourproject
   ```

3. Open the `login.html` file in your favorite web browser to view and interact with the login page.

## Usage

Once you have opened the `login.html` file in a web browser, you will see a login form. You can enter your email and password and click the "Sign in" button. The interface is styled to provide a visually appealing experience, utilizing Tailwind CSS for layout and design.

## Features

- Responsive design that adapts to different screen sizes.
- User-friendly form with fields for email and password.
- Icons for input fields using Font Awesome.
- A "Remember me" checkbox for user convenience.
- Links for password recovery.

## Dependencies

This project uses the following dependencies:

- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/) - A popular icon set and toolkit.
- Google Fonts: [Inter](https://fonts.google.com/specimen/Inter) - Used for the font styling.

No additional libraries are required, and all dependencies are included via CDN links in the HTML file.
